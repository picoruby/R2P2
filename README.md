## Quick start (new location)

```sh
git clone https://github.com/picoruby/picoruby.git
cd picoruby
bundle install
rake r2p2:setup
rake r2p2:picoruby:pico:debug
```

See the [picoruby-r2p2 README](https://github.com/picoruby/picoruby/blob/master/mrbgems/picoruby-r2p2/README.md) for details.
